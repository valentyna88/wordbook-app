# 📚 WordBook - Vocabulary Learning App

Mobile app for learning and managing vocabulary.

## Tech Stack

- React Native
- Expo
- TypeScript
- Expo Router

## Features (in progress)

- Add new words
- View list of words
- Edit words
- Delete words

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npx expo start
```

## Project Structure

- `app/` - screens and navigation (Expo Router)
- `components/` - reusable UI components
- `assets/` - images and icons
